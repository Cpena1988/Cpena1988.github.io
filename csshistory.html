Front-End Design
- Front End Design refers to the visual layer of websites and applications
- This is typically the UI, layout, typography, images, and other visual elements and their styling
- Can often refer to usability and user experience design as well
- The most common languages for creating front end content are HTML, CSS, and JavaScript
- Typical front-end design roles include:
1. Creating mockups & develping visual standards
2. Structuring semantic content
3. Handling site accessibility
4. Controlling site typography
5. Page layout
6. Form design
7. Interactivity & animation
8. Creating site imagery
