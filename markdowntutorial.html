Dealing With Data
- HTML5 introduced the web storage API
- Allowing web sites to store key value pairs
- A key value pair is a set of values that make it easier for a browser to save user preferences on sites
- XML stands for extensible markup language
- JSON stands for JavaScript Object Notation and allows you to store simple values in a text file
- The most efficient way to store complex data is to use a database
- There are currently two methods that allow you to store databases within the browser Indexed DB & Web SQL
- Relational databases store data in tables that relate to each other based on item key values
- Relational databases include Oracle, mySQL, Sybase, and other SQL-based databases
- Non-Relational databases store data individual flat, object-driven datasets
- These are often faster to index, and scale easier than relational databases
- Non-Relational databases include MongoDB, CouchDB, SImpleDB, HBase, CouchBase, and Cassandra
