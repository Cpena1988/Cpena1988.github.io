Other Popular Server Languages
- JSP stands for Java Server Pages it is a server side application tool developed by Sun
- JSP code is embedded directly into the HTML, and then processed on the server
- Scripts are combinations of XML and Java scriptlets which can be difficult for beginners
- JSP is widely used in enterprise-level sites, but can be used for any sized site or application
- It's easy to find hosting, but choose the proper Java platform
- (Visual Basic) VB .NET is part of the larget .NET Framework
- Often used for large-scal, enterprise sized sites or applications
- Can only be hosted on Windows servers
- ColdFusion uses a markup language much like HTML
- A central administrator makes many tasks simple
- ColdFusion sites and applications often require less code than other languages
- It is harder to find and more expensive
- Python is a powerful development language that has a variety of web-related libraries
- Typically used to create larger sites or applications
- Ruby is a general-purpose development language
- Rails is an extension of Ruby and has a simple syntax and powerful features, a popular choice for creating dynamic websites
- Most designers choose a platform based on cost, developer network, and ease of use
