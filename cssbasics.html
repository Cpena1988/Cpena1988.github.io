Content Delivery Networks
- CDN stands for Content Delivery Netowork and is a system for delivering content over distributed network of servers
- CDNS are largely used to serve static resources such as javascript libraries, video, or other site dependencies
- Using Google's Hosted Libraries reduces load from your server, speed the delivery of jQuery to the client, and increases the liklihood that the client will have jQuery cached
- Some CDNs host a single type of resource
- CDNs like YouTube serve not only as delivery networks, but content aggregators as well
- Popular CDNs include Akamai, CloudFare, Amazon's CloudFront, EdgeCast, and Windows Azure
- CDN Issues include:
1. Adds an addtional layer of complexity to your site
2. For dedicated resources, you don't have the ability to create custom downloads or specific minified version
3. Certain security requirements might prevent the use of an external CDN
